# Windows Subsystem for Linux

Windows Subsystem for Linux (WSL) is a compatibility layer for running Linux binary executables (in ELF format) natively on Windows 10, Windows 11, and Windows Server 2019. Read more on [Wikipedia](https://en.wikipedia.org/wiki/Windows_Subsystem_for_Linux).
