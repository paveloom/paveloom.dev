PMG #

pmg.paveloom.dev
PMG is a project about creating an algorithm for spatial-kinematic modelling of a flat subsystem of the Milky Way by the method of maximum likelihood. The provided binary crate enables a user to infer the parameters of the Galaxy by optimizing the model, with masers serving as reference objects.
